CALLBACK:
A CALLBACK IS A FUNCTION PASSED AS AN ARGUMENT TO ANOTHER FUNCTION, WHICH IS THEN INVOKED INSIDE THE OUTER FUNCTION TO COMPLETE SOME KIND OF ROUTINE OR ACTION.

CALLBACK HELL:
CALLBACK HELL REFERS TO A SITUATION IN ASYNCHRONOUS PROGRAMMING WHERE CALLBACKS ARE NESTED WITHIN OTHER CALLBACKS MULTIPLE LEVELS DEEP, MAKING CODE DIFFICULT TO READ, MAINTAIN, AND DEBUG. THIS OFTEN RESULTS IN A "PYRAMID" OR "ARROW" STRUCTURE IN THE CODE.
